# 🚗 Vehicle Rental Management System

## 📌 Introduction
Vehicle Rental Management System is a full-stack web application that allows users to browse, book, and manage vehicle rentals online. It also provides an admin panel to manage vehicles, bookings, and users efficiently.

---

## ✨ Features
- 🔐 User Authentication (Login / Signup)
- 🚘 Browse Available Vehicles
- 📅 Book Vehicles Online
- 🧑‍💼 Admin Dashboard
- 🛠️ Add / Edit / Delete Vehicles
- 📊 Manage Bookings & Vehicles

---

## 🛠️ Tech Stack
**Frontend:**
- React.js
- HTML, CSS, JavaScript

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

---

vehicle-rental-system/

## 🚀 Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/Aminkhan12631/vehicle-rental-system.git


### 2️⃣ Install dependencies

👉 For frontend:

cd client
npm install


👉 For backend:

cd server
npm install


---

### 3️⃣ Run the project

👉 Start backend:

cd server
node server.js


👉 Start frontend:

cd client
npm run dev
